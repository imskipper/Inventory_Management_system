## üìñ Overview
The **Inventory Management System** is a web-based application designed to streamline the process of tracking stock, managing suppliers, and monitoring sales. Developed as a **DBMS Final Project**, this system ensures data integrity and provides an intuitive interface for store managers to handle day-to-day operations efficiently.

## ‚ú® Key Features
* **Dashboard:** Real-time overview of total items, sales, and low-stock alerts.
* **Product Management:** Add, update, delete, and categorize inventory items.
* **Supplier Management:** Maintain records of suppliers and purchase history.
* **Stock Tracking:** Automated updates to stock levels upon sales or purchases.
* **Reporting:** Generate basic reports on inventory status and sales performance.
* **Authentication:** Secure login system for administrators.
